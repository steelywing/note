## Help
```powershell
Get-Help [<cmdlet>] [-ShowWindow] [-Detailed|-Full|-Examples]
```

## Alias
```powershell
Get-Alias [<alias>]
```

## Filter string
```powershell
Select-String "<pattern>"
```

## Network profile
```powershell
Get-NetConnectionProfile
Set-NetConnectionProfile (-InterfaceIndex <ID>|-Name <name>) -NetworkCategory (Private|Public)
```
