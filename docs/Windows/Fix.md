# Fix - Windows

Fix / Troubleshoot

## Microsoft.Windows.Photos with high CPU usage

- `Default apps`
  - Disable `Photos` app
- Reset `Photos` app

## Get service of `svchost.exe`

- Right click `svchost.exe` process in Task Manager
- Click "Go to Service(s)"

## Windows Loader

Windows Loader does not work on UEFI
