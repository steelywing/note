# DNS

```
Switch(config)# ip name-server <DNS IP address> ...
```

## Disable domain name lookup in global mode

```
Switch(config)# no ip domain-lookup
```

## Device name

```
Switch(config)# hostname <host name>
```

## Device domain

```
Switch(config)# ip domain-name <domain>
```
