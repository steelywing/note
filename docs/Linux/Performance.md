# Performance

## Display memory info

```bash
free <options>
```

| Option | Description |
| - | - |
| `-h` | Human readable unit |

## Display process info

```bash
# BSD syntax
ps [aux]

# Unix syntax
ps [-ef]
```

| Option | Description |
| - | - |
| `-e` \| `-A` | Entire / All processes |
| `-f` | Full format / Detail |
