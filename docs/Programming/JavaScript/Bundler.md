# Bundler

## WebPack

- For JS, CSS, HTML
- Popular

## Parcel

- For JS, CSS, HTML
- Zero configuration

## Rollup

- For JS
- Plugin for JSX
- Faster
- Support `import` ES6, CommonJS, ...
- Support output format AMD, CommonJS, ES, IIFE, UMD, SystemJS

## ESBuild

- For JS, JSX
- Extremely fast
- Written in Go

## Vite

- For JS, CSS, HTML
- Faster
- Using Rollup for Production
- Using esbuild for Dev
