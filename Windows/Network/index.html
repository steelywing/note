Reserve TCP/UDP port#

Reference

Add / Delete reserved port

netsh int[erface] ipv4 `    { add | delete } excludedportrange `    { tcp | udp } `    <start port> `    <number of ports> `    [ active | persistent ]

Show reserved port

netsh int[erface] ipv4 `    show excludedportrange { tcp | udp }
