Computer Management / Local User#

lusrmgr.msc

Create User#

net user <user name> [<password>] /add
# PowerShellNew-LocalUser <user name> [-Password (Read-Host -AsSecureString) | -NoPassword]

Delete User#

net user <user name> /delete
# PowerShellRemove-LocalUser <user name>

Make password never expire#

wmic UserAccount [where Name=<user name>] set PasswordExpires={True|False}

Using group policy

Computer Configuration > Windows Settings > Security Settings > Password Policy > Maximum password age > Set to 0

Cortana#

Disable Cortana in Windows 10

reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows\Windows Search" /v "AllowCortana" /t REG_DWORD /d 0 /f
## PowerShellNew-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\Windows Search" -Name "AllowCortana" -PropertyType DWord -Value 0 -Force

Enable Cortana in Windows 10

reg delete "HKLM\SOFTWARE\Policies\Microsoft\Windows\Windows Search" /v "AllowCortana" /f
## PowerShellRemove-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\Windows Search" -Name "AllowCortana"
