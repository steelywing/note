房屋​

Ref

  • ≥ 60 格（包括牆）
  • 入口：門 / 機關門 / 平台
  • 光源：火把 / 燈
  • 平坦表面：工作台、桌
  • 舒適性：椅子、床
  • 背景牆
  • 最少有 1 格 實心地板（非平台）給 NPC 站立
    • 實心地板 不可與 兩側牆 或 開啟的門 相鄰
  • 有 2×3 格的空間
    • 可以與 平坦 或 舒適性 物品重疊
    • 不能與 光源 重疊
