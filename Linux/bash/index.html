  <...>

  // $i == <current value>
done

Search history​

Ctrl + R

Remove current session bash history​

Clear current history buffer

history -c

Don't save the history when session close

unset HISTFILE

Kill bash

## $$ == Current shell PID
kill -9 $$

Delete all bash history​

Empty history file

> $HISTFILE

HISTSIZE=0
