IP forward#

Reference

Get IP forward setting

sysctl net.ipv4.ip_forward

Set IP forward setting (0 = Disable, 1 = Enable)

sysctl -w net.ipv4.ip_forward={0|1}

Set IP forward on boot

/etc/sysctl.conf

net.ipv4.ip_forward = {0|1}

Traffic control#

  • Reference
  • Manual
  • man tc-netem

Show qdisc (queueing discipline)

tc qdisc show dev <dev>

Remove qdisc (queueing discipline)

tc qdisc add dev <dev> root

netem (Network Emulator)#

Latency

tc qdisc add dev <dev> root netem \  delay <min latency>ms [<max latency>ms]

Loss / Corrupt / Duplicate

tc qdisc add dev <dev> root netem \  { loss | corrupt | duplicate } <chance>%

TBF (Token Bucket Filter)#

Limit traffic rate / speed / bandwidth

# min burst = rate / HZ(1000)# limit = queue size# latency = max amount of time in queue
tc qdisc add dev <dev> root tbf \  rate <rate>{k|m}bit \  burst <burst>{k|m}bit \  { limit <limit>{k|m}bit | latency <latency>ms }
