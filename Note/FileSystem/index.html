exFAT#

  • No journaling
  • Max volume size 128 PiB
  • Max file size 16 EiB
  • No ACL

FAT32#

  • No journaling
  • Max volume size 2 TiB (4 KiB Cluster Size)
  • Max file size 4 GiB
  • No ACL

NTFS#

  • Journaling
  • Max volume size 16 TiB (4 KiB Cluster Size)
  • Max file size 16 EiB
  • ACL

BTRFS#

  • Unstable
  • Copy-on-Write
  • Snapshot
  • RAID
  • Slow

XFS#

  • Journaling
  • RedHat
  • Cannot shrinking

ZFS#

  • Copy-on-Write
  • Snapshot
  • RAID
